# NewsPulse - Smart News Scraper

NewsPulse is an industry-level Python based web scraping project that collects latest news headlines from multiple sources.

# Features

- Web scraping using Python
- Robots.txt compliance
- Keyword based filtering
- Duplicate headline removal
- Alphabetical sorting
- Logging system
- Export data into JSON and CSV

#Tech Stack

- Python
- BeautifulSoup
- Requests
- JSON
- CSV
- Logging

# Project Structure

main.py
scraper.py
logger.py
config.py
utils.py

# Output

Generated files:
- headlines.json
- headlines.csv

python main.py
